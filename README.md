# ☁️ KLY – Minimal Weather Checker

A clean and minimalist web app that fetches and displays weather data using HTML, CSS, and JavaScript.

## 💡 Features
- Live weather updates by city
- Responsive UI
- Minimalistic design

## 🌐 Built With
- HTML5, CSS3
- JavaScript
- OpenWeatherMap API (optional)
